# NEW MEDIA — وكالة تسويق شاملة

موقع وكالة **NEW MEDIA**، وكالة تسويق شاملة مقرها جدة، المملكة العربية السعودية.

---

## نبذة عن المشروع

موقع ترويجي أحادي الصفحة (Landing Page) يعرّف بالوكالة وخدماتها ويشجّع الزوار على التواصل. الموقع مبني بـ HTML/CSS/JS خالص دون أي فريموورك، ومستضاف على GitHub Pages.

## الأقسام

| القسم | الوصف |
|---|---|
| Hero | تعريف بالوكالة ودعوة للتواصل |
| أقسامنا | عرض الخدمات الرئيسية |
| أعمال مختارة | معرض لبعض المشاريع المنجزة |
| تواصل معنا | CTA للتواصل المباشر |

## الخدمات

- **قسم الهوية والتجربة** — هوية بصرية، تصميم، تطوير مواقع ومتاجر إلكترونية
- **الإنتاج الإعلامي** — تصوير، فيديو، إنتاج إعلانات، وفيديو بالذكاء الاصطناعي

## الصفحات

```
index.html            — الصفحة الرئيسية
404.html              — صفحة الخطأ
privacy-policy.html   — سياسة الخصوصية
cookie-policy.html    — سياسة ملفات تعريف الارتباط
confidentiality.html  — سياسة السرية
refund-policy.html    — سياسة الاسترداد
terms-of-service.html — شروط الخدمة
```

## التقنيات

- HTML5 / CSS3 / Vanilla JavaScript
- خط Alexandria و Tajawal من Google Fonts
- تصميم RTL كامل (اللغة العربية)
- ثيم داكن مع هوية بصرية بلونَي البنفسجي `#6600cc` والبرتقالي `#ff6600`
- Scroll animations باستخدام IntersectionObserver

## التشغيل محلياً

لا يحتاج الموقع إلى build steps — افتح `index.html` مباشرة في المتصفح أو شغّل سيرفر محلي بسيط:

```bash
npx serve .
# أو
python -m http.server 8080
```

## الاستضافة

الموقع مستضاف على **GitHub Pages** ويُنشر تلقائياً من branch الـ `main`.
